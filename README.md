Barotrauma mod to play MIDI files on instruments
Steam workshop link - https://steamcommunity.com/sharedfiles/filedetails/?id=3692939185


in init file MidiMod.Debug = true
If true your midi files will be searching in %moddir%/Midi
And activate some console debugging

If false no debug and midis in this worshop folder (item - https://steamcommunity.com/workshop/filedetails/?id=3695216167)

For example on windows
C:\Users\User\AppData\Local\Daedalic Entertainment GmbH\Barotrauma\WorkshopMods\Installed\3695216167
and linux
~/.local/share/Daedalic Entertainment GmbH/Barotrauma/WorkshopMods/Installed/3695216167/Midi
